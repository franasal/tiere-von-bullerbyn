# Foto-Helfer Anleitung

Wir brauchen Fotos von den Merkmalen der Bullerbyn-Tiere, damit Besucher:innen sie besser erkennen können. So kannst du helfen:

## Für Helfer:innen

### 1. Helfer-Modus öffnen

Du bekommst einen Link von der Admin-Person (z.B. `https://deine-seite.de/tiere-von-bullerbyn/?helper`).
Öffne den Link auf deinem Handy.

### 2. Was du siehst

- Eine Liste aller Tiere, bei denen noch Fotos fehlen
- Für jedes Tier: welche Merkmale fotografiert werden müssen
- z.B. „Ohrmarke: Rechts", „Schwanz: Kurz", „Flecken: Schwarze Flecken im Gesicht"

### 3. Foto aufnehmen

1. Tippe auf ein Tier, um die fehlenden Merkmale zu sehen
2. Tippe auf **„Foto aufnehmen"** — die Kamera öffnet sich
3. Fotografiere das Tier so, dass das jeweilige Merkmal **gut sichtbar** ist
4. Tippe auf **„Foto teilen"** — das Foto wird per WhatsApp, E-Mail oder einer anderen App geteilt

### Tipps für gute Fotos

- **Ohrmarke/Loch**: Nah an den Kopf herangehen, Ohr gut sichtbar
- **Schwanz**: Seitlich oder von hinten fotografieren
- **Flecken/Fellfarbe**: Ganze Körperseite zeigen
- **Besondere Merkmale** (Buckel, Stoßzähne, Haut): Gezielt dieses Merkmal ins Bild rücken
- Tageslicht ist am besten, kein Blitz

## Für Admins

### Admin-Modus aktivieren

- **5x auf den Titel** „Die Tiere von Veganien" tippen, oder
- `?admin` an die URL anhängen (z.B. `https://deine-seite.de/tiere-von-bullerbyn/?admin`)

### Was der Admin-Modus kann

- **Übersicht**: Alle Tiere mit ihren Merkmalen, grün/rot markiert ob Foto vorhanden
- **Filter**: Nur fehlende, nur vorhandene, oder alle anzeigen
- **Fotos hochladen**: Pro Merkmal entweder direkt fotografieren oder eine URL einfügen
- **Helfer-Link teilen**: Generiert einen Link zum Helfer-Modus, den du per WhatsApp/E-Mail verschicken kannst
- **Export/Import**: Alle Foto-Zuordnungen als JSON exportieren oder importieren (für Backups oder Geräte-Transfer)

### Fotos von Helfern einpflegen

1. Helfer:innen teilen Fotos per WhatsApp/E-Mail
2. Lade die Fotos auf die Website hoch (z.B. WordPress Mediathek) oder in Google Photos
3. Kopiere die URL des hochgeladenen Bildes
4. Öffne den Admin-Modus → Tier aufklappen → beim richtigen Merkmal die URL einfügen

### Speicherung

- Fotos/URLs werden im **localStorage** des Browsers gespeichert (nur auf diesem Gerät)
- Nutze **Export/Import** um Daten zwischen Geräten zu übertragen
- localStorage hat ein Limit (~5 MB) — bei vielen Fotos ist die URL-Methode empfohlen

## Welche Merkmale werden gebraucht?

Jedes Tier hat spezifische Merkmale, die es von den anderen unterscheiden. Pro Tier werden typischerweise 3-6 Merkmal-Fotos gebraucht:

| Merkmal | Beschreibung |
|---------|-------------|
| Typ | Hausschwein, Hängebauchschwein, Wollschwein, Wildschwein |
| Geschlecht | Männlich / Weiblich (nur bei Hausschweinen) |
| Fell/Borsten | Farbe und Art des Fells |
| Flecken | Muster und Position von Flecken |
| Ohren | Hängend oder aufrecht |
| Schwanz | Kurz, lang mit/ohne Haare, geringelt |
| Ohrmarke | Links, rechts, oder keine |
| Ohrspitzen | Vorhanden oder fehlend |
| Hautfarbe | Rosa, etc. |
| Rücken | Besondere Färbung |
| Haut | Schuppig oder normal |
| Größe | Groß oder klein |
| Stoßzähne | Sichtbar oder nicht |
| Buckel | Vorhanden oder nicht |
| Körperbau | Auffällige Körperform |
