# Pig Cue Images

Lege Frage-bezogene Hinweisfotos in den Unterordnern pro Schwein ab.

Ordnerstruktur:
- `src/assets/pig-cues/<PigName>/...`

Dateinamen-Hinweis:
- Wenn moeglich, den Trait-Key im Dateinamen nutzen, damit Bilder zur aktuellen Frage priorisiert werden.
- Beispiele:
  - `tailType_1.jpg`
  - `earForm_closeup.png`
  - `earMark_right.jpeg`
  - `spotPattern_face_body.webp`

Unterstuetzte Formate:
- `.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`
